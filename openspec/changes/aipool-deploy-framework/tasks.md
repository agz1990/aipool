1. **SSH 连接稳定性**：可能需要更多时间处理网络问题
2. **adapter.yaml 复杂度**：实际使用中可能发现需要更多功能
3. **用户体验**：manual 步骤的交互可能需要多次迭代
4. **测试覆盖**：完整测试所有场景可能比预期耗时
